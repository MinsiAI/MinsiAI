# Minsi

I am a student builder at Guangzhou Foreign Language School working on Minsi.ai, a privacy-first AI companion for emotional reflection, self-understanding, and safer digital interaction.

Most of my current work is around affective computing, full-stack web products, and AI systems that respect privacy by default.

## Current Work

### Minsi.ai

Minsi.ai helps students reflect on stress, emotions, and online identity through text and voice interaction, research feedback, and privacy-centered safety design.

- Live demo: https://minsi.ai
- Project overview: [PROJECT_README.md](PROJECT_README.md)
- Stack: Next.js, TypeScript, Java Spring Boot, MySQL, Redis, AI provider abstraction
